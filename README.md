## 🚀 Featured Project

### [Arbit](https://arbit-finance.vercel.app/)
**Real-Time Financial Data Pipeline — Kafka · Redis · PostgreSQL · Node.js**

A multi-service distributed pipeline that ingests live OHLCV stock data, computes technical indicators (SMA, EMA, RSI), detects anomalies, and persists results to PostgreSQL via Prisma.

### 🏗️ Architecture

* [**s_1**](https://github.com/mdrayaanpasha/FIN_PIP_SERVER_1) — Yahoo Finance producer → Kafka (Aiven, SSL/mTLS)
* [**s_2**](https://github.com/mdrayaanpasha/FIN_PIP_SERVER_2) — Anomaly detection consumer
* [**s_3**](https://github.com/mdrayaanpasha/FIN_PIP_SERVER_3) — Technical indicators: SMA, EMA, RSI
* [**s_4**](https://github.com/mdrayaanpasha/FIN_PIP_SERVER_4) — PostgreSQL persistence via Prisma 6
* **Storage** — Redis for inter-service data; Kafka for event fan-out
